# 🎬 Netflix Movies & TV Shows SQL Data Analysis

## 📌 Project Overview

The main objective of this project is to solve real-world business questions, explore content trends, and extract meaningful insights from Netflix data.

# 📊 Business Problems Solved

### 1. Movies vs TV Shows Analysis
Calculated the total number of Movies and TV Shows available on Netflix.

### 2. Most Common Ratings
Identified the most frequently used ratings for Movies and TV Shows.

### 3. Content Release Analysis
Analyzed movies released in specific years.

### 4. Country-wise Content Analysis
Found the top countries producing the highest Netflix content.

### 5. Longest Movie Analysis
Identified movies with the highest duration.

### 6. Recent Content Analysis
Found content added to Netflix in recent years.

### 7. Director Analysis
Analyzed Movies and TV Shows created by specific directors.

### 8. TV Shows Season Analysis
Found TV Shows having more than 5 seasons.

### 9. Genre Analysis
Analyzed different Netflix categories and their content count.

### 10. Country-based Release Trends
Calculated average release years for specific countries.

### 11. Documentary Analysis
Filtered documentary movies available on Netflix.

### 12. Missing Data Analysis
Identified records with missing director information.

### 13. Actor Content Analysis
Analyzed movies featuring specific actors.

### 14. Top Actor Analysis
Found actors appearing most frequently in Indian Netflix movies.

### 15. Content Categorization
Classified content based on keywords present in descriptions.
